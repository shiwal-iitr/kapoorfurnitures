# Furniture Store - Casa Elegante

A premium furniture showcase website built with Next.js 15, React 19, and Tailwind CSS v4.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- 🏠 Homepage with hero section and featured products
- 🛍️ Product catalog with 54+ products
- 🔍 Category filters (Beds, Almirahs, Sofa Sets)
- 📱 Mobile responsive design
- ⚡ SEO optimized with JSON-LD structured data
- 💬 WhatsApp integration
- 🎨 Premium dark theme with gold accents
